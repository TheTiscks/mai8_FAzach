## Списочные структуры данных. Очереди и стеки. Организация очереди и стека на основе односвязного списка. Вычислительные сложности операций доступа к элементу, вставки и удаления элементов для стека и очереди, их реализация на языке C. Алгоритм проверки баланса скобок в выражении.

Очереди и стеки — это абстрактные типы данных (АТД), которые представляют собой коллекции элементов, доступ к которым осуществляется по определенным правилам.

### Очереди
Очередь (queue) — это структура данных, работающая по принципу "первым пришел
— первым ушел" (FIFO - First In, First Out). Элементы добавляются в конец очереди
(enqueue) и удаляются из начала (dequeue)

* enqueue(element): Добавляет элемент в конец очереди.
* dequeue(): Удаляет и возвращает элемент из начала очереди.
* front(): Возвращает элемент из начала очереди без удаления.
* isEmpty(): Проверяет, пуста ли очередь.
* size(): Возвращает количество элементов в очереди.

````C
typedef struct {
    Node *head;
    Node *tail;
} Queue;

void queue_enqueue(Queue *q, int value) {
    Node *n = malloc(sizeof(Node));
    n->data = value;
    n->next = NULL;
    if (q->tail == NULL) {
        q->head = q->tail = n;
    } else {
        q->tail->next = n;
        q->tail = n;
    }
}

int queue_dequeue(Queue *q) {
    Node *temp = q->head;
    int value = temp->data;
    q->head = q->head->next;
    if (q->head == NULL) q->tail = NULL;
    free(temp);
    return value;
}
````


### Стек
Стек (stack) — это структура данных, работающая по принципу "последним пришел —
первым ушел" (LIFO - Last In, First Out). Элементы добавляются и удаляются с
вершины стека.

Операции:
* push(element): Добавляет элемент на вершину стека.
* pop(): Удаляет и возвращает элемент с вершины стека.
* peek(): Возвращает элемент с вершины стека без удаления.
* isEmpty(): Проверяет, пуст ли стек.
* size(): Возвращает количество элементов в стеке

````C
typedef struct {
    Node *top;
} Stack;

void stack_push(Stack *s, int value) {
    Node *n = malloc(sizeof(Node));
    n->data = value;
    n->next = s->top;
    s->top = n;
}

int stack_pop(Stack *s) {
    Node *temp = s->top;
    int value = temp->data;
    s->top = temp->next;
    free(temp);
    return value;
}
````


### Операция size для стека и очереди
##### Теория: операция size возвращает количество элементов в структуре данных.
Для стека и очереди на основе односвязного списка сложность O(n), если просто проходить по списку (неэффективно для больших структур).
Чтобы сделать O(1), добавляем счётчик в структуру (size_t count) и обновляем его при каждой вставке/удалении.

Для стека, добавляем в структуру: 
````C
typedef struct {
    Node *top;
    size_t count;    // для O(1) size
} Stack;
````

Инициализация: 
````C
s->count = 0;
push: s->count++;
pop: s->count--;
size: return s->count;   // O(1)
````



#### Для очереди
Аналогично добавляем count в Queue.

````C
enqueue: q->count++;
dequeue: q->count--;
size: return q->count;   // O(1)
````
Без счётчика — O(n) проход от head до NULL.
