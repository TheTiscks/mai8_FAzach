## Приоритетные очереди. Биномиальное дерево, ранг биномиального дерева, число элементов в биномиальном дереве заданного ранга. Биномиальная и фибоначчиева приоритетные очереди, их реализация на базе структур данных вида кольцевой список и дерево общего вида. Операции вставки элемента, поиска минимума/максимума, удаления минимума/максимума, слияния с разрушением и без разрушения исходных биномиальных/фибоначчиевых приоритетных очередей; их вычислительные сложности и реализация на языке C.

### Биномиальное дерево
Биномиальное дерево ранга k — это дерево, состоящее из корня и k поддеревьев рангов 0, 1, …, k−1.

Число элементов в биномиальном дереве ранга k равно точно 2^k.

Ранг — это количество детей корня (и одновременно логарифм числа узлов).
### Биномиальная куча (binomial heap)
Это лес из биномиальных деревьев разных рангов с разными корневыми ключами, удовлетворяющих свойству кучи (min-heap или max-heap).

Корни деревьев связаны в кольцевой односвязный список (обычно по возрастанию ранга).
### Фибоначчиева куча (Fibonacci heap)
Это тоже лес деревьев общего вида, удовлетворяющих свойству кучи. Деревья связаны в кольцевой двусвязный список корней.

Дополнительные свойства:
* степень узла ограничена O(log n)
* пометка удалённых детей для отложенной очистки

### Операции и их вычислительные сложности:

* Поиск минимума — O(1) для обеих (указатель на минимальный корень или проход по списку корней)
* Вставка — O(log n) для биномиальной, амортизированная O(1) для фибоначчиевой
* Удаление минимума — O(log n) для биномиальной, амортизированная O(log n) для фибоначчиевой
* Слияние с разрушением — O(log n) для биномиальной, амортизированная O(1) для фибоначчиевой
* Слияние без разрушения — O(log n) для обеих (копирование списка корней или пути слияния)

### Реализация на C — основные идеи и структуры
#### Биномиальная куча
````C
typedef struct Node {
    int key;
    int degree;                // ранг = число детей
    struct Node *child;        // первый ребёнок
    struct Node *sibling;      // следующий брат
    struct Node *parent;       // для удобства (не обязательно)
} Node;

typedef struct {
    Node *head;                // кольцевой список корней (по возрастанию degree)
} BinomialHeap;
````
Слияние — ключевая операция: сливаем корни как бинарные числа (по рангам), снося «перенос» при одинаковых рангах.
#### Фибоначчиева куча
````C
typedef struct Node {
    int key;
    int degree;
    int mark;                  // пометка потерянного ребёнка
    struct Node *parent;
    struct Node *child;        // первый ребёнок (кольцевой список детей)
    struct Node *left;         // предыдущий корень в кольцевом списке
    struct Node *right;        // следующий корень
} Node;

typedef struct {
    Node *min;                 // указатель на корень с минимальным ключом
    size_t n;                  // число элементов
} FibonacciHeap;
````

Слияние — просто соединяем кольцевые списки корней и обновляем min (амортизированная O(1)).
Удаление минимума — добавляем детей минимума в список корней + консолидация одинаковых степеней
