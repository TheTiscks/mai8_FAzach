## Функции. Способы передачи аргументов в функции. Возврат значений из функции через возвращаемое значение, через параметры функции. Глобальные и локальные переменные. Область видимости переменных. Модификатор static.

#### Функция – это блок кода, который выполняет определенную задачу.
Функции позволяют разбить программу на более мелкие, управляемые части, повышая
читаемость, повторное использование кода и модульность.

Объявление функции:
``тип_возвращаемого_значения имя_функции(тип_аргумента1 аргумент1, тип_аргумента2 аргумент2, ...);``

### Способы передачи аргументов

* По значению — изменения, сделанные с копией внутри функции, не влияют на исходный аргумент
* Имитация передачи по ссылке — передаём указатель и разыменовываем внутри. Входные данные могут изменяться!

### Возврат значений из функции

* Через return — одно значение любого типа (кроме массива и функции)
* Через параметры-указатели — для возврата нескольких значений одновременно

### Области видимости и время жизни переменных
Локальная — объявлена внутри блока {} — видима от точки объявления до конца блока — живёт до выхода из блока

Параметр функции — в списке параметров — виден только внутри функции — живёт до возврата из функции

Глобальная — объявлена вне всех функций — видима во всём файле после объявления и в других файлах через extern — живёт всю программу

Статическая локальная — объявлена внутри функции или блока с ключевым словом static — видима только внутри этого блока — живёт всю программу, сохраняет значение между вызовами

Статическая глобальная переменная — объявлена вне функций с static — видима только в текущем .c-файле — живёт всю программу

Статическая глобальная функция — объявлена с static — видима только в текущем .c-файле — живёт всю программу

### 4 основных применения модификатора static в языке C

* static для локальной переменной (самое частое на зачёте)
Переменная сохраняет своё значение между вызовами функции.
Инициализируется один раз (при старте программы или при первом входе в блок).

* static для глобальной переменной
Ограничивает область видимости текущим файлом перевода (translation unit).
Другие .c-файлы не видят её даже через extern.
* static для функции
Делает функцию видимой только внутри текущего .c-файла.
Полезно для скрытия вспомогательных функций от остального проекта.
* static для члена структуры или объединения (с C11)
Создаёт фиксированный (статический) член, общий для всех экземпляров структуры.
Аналогично static-членам в C++.
