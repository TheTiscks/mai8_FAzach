## Структуры и объединения в языке C, их размер, определение и особенности использования. Оператор “.”. Указатели на структуры в языке C, оператор “->”

### Структура (struct)
Структура — это пользовательский тип данных, который объединяет несколько переменных (членов) разных типов под одним именем.
Члены хранятся в памяти последовательно (с возможным выравниванием — padding).

#### Определение:
Объявление типа: struct Tag { int a; double b; char c; };
Создание объекта: struct Tag obj;
Анонимная структура (редко): struct { ... } obj;

Размер структуры
sizeof(struct Tag) ≥ сумма размеров членов.

Компилятор добавляет padding для выравнивания членов по границам, соответствующим их типу (обычно по 4 или 8 байтам на x64).

Последний член может иметь padding до размера самого большого члена (для массивов структур).

Чтобы минимизировать padding — располагать члены от большего типа к меньшему.

#### Особенности использования

* Доступ к члену: оператор «.» (точка) — obj.member
* Можно копировать структуры присваиванием: struct1 = struct2; (побитовое копирование)
* Массив структур хранится плотно (с учётом padding одного элемента)
* Гибкий член массива (flexible array member, с C99): последний член char data[]; — размер структуры не включает его, выделяется динамически.

### Объединение (union)
Объединение — тип данных, где все члены занимают одну и ту же область памяти (начиная с одного адреса).

Размер union равен размеру самого большого члена (плюс возможный padding до выравнивания).

Смысл: экономия памяти или интерпретация одних и тех же байтов разными типами (например, типизированный вариант void*).

Определение аналогично struct, но ключевое слово union.

Доступ к члену тоже через «.»

В любой момент времени «активен» только последний записанный член — чтение другого даёт неопределённое поведение (но на практике работает для low-level).

### Указатели на структуры

Указатель на структуру объявляется как struct Tag *p;
* Можно указывать на объект: p = &obj;
* Или на динамически выделенную: p = malloc(sizeof(struct Tag));

### Оператор «->»
Сокращение для доступа к члену через указатель: p->member

Эквивалентно (*p).member

! Обязательно использовать -> с указателями, иначе синтаксическая ошибка.
